# Countries Front-End
<a><img width="100%" src="https://github.com/Audvis/countriesFront/blob/main/utilities/countriesPC1.png"></a>

https://countriesfront.herokuapp.com/

Proyecto realizado como parte de la evaluación para aprovar el bootcamp de desarrollo web full stack [Soy Henry](https://www.soyhenry.com/)

## Tecnologías utilizadas :computer: 

* __Javascript__
* __React - Redux__
* __Hooks__


## Instalación :feet:

```
git clone https://github.com/Audvis/countriesFront.git
npm install >> countriesFront/
npm start >> countriesFront/

```
